# yuexboot-mall

## 项目简介

yuexboot-mall 是一套基于 Spring Boot 3.x 开发的 H5 商城系统，采用前后端分离架构。项目包含运营后台、H5 商城前台和后端接口服务，实现了一套完整的电商业务流程。

### 核心功能

- **商品展示**：首页推荐、商品分类、商品详情、SKU 选择
- **搜索功能**：关键词搜索、历史搜索记录、热门搜索
- **购物车**：添加商品、数量调整、批量结算
- **订单管理**：下单流程、订单列表、订单状态追踪
- **支付功能**：支持支付宝、微信支付、易支付等多种支付方式
- **用户中心**：个人信息管理、收货地址管理
- **商品评价**：订单评论、评价查看
- **后台管理**：商品管理、订单管理、用户管理、权限控制等

### 技术特点

- 完整的电商业务闭环
- 前后端分离架构
- RESTful API 设计
- 支持高并发场景
- 易于本地开发和部署

| **监控** | Spring Boot Admin 3.1.0 | 应用监控管理 |

### 前端技术

#### H5 商城前台
- **框架**：Vue 2.x
- **UI 组件库**：Vant UI
- **状态管理**：Vuex
- **路由**：Vue Router
- **HTTP 客户端**：Axios

#### 管理后台
- **框架**：Vue 2.x
- **UI 组件库**：Element UI
- **状态管理**：Vuex
- **路由**：Vue Router

### 其他中间件

- **Nginx**：反向代理、负载均衡
- **Docker**：容器化部署
- **Docker Compose**：多容器编排

## 项目结构

yuexboot-mall/ ├── yuexboot-common # 通用模块（核心基础类、entity、mapper、service） ├── yuexboot-admin-api # 运营后台 API 模块 ├── yuexboot-mobile-api # H5 商城 API 模块 ├── yuexboot-data # 数据访问模块 │ ├── yuexboot-data-redis # Redis 配置模块 │ └── yuexboot-data-elastic # Elasticsearch 配置模块 ├── yuexboot-message # 消息队列模块 │ ├── yuexboot-message-core # RabbitMQ 配置 │ └── yuexboot-message-consumer # 消费者模块 ├── yuexboot-job # 定时任务模块 ├── yuexboot-monitor # 监控模块 ├── yuexboot-util # 工具类模块 └── db-init # 数据库初始化脚本

## 系统架构

项目采用经典的分层架构设计：

- **表现层**：Controller 层，处理 HTTP 请求
- **业务层**：Service 层，实现业务逻辑
- **持久层**：Mapper 层，数据访问操作
- **基础设施层**：Redis、Elasticsearch、RabbitMQ 等中间件集成
